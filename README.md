# Move Car Page

一个纯静态的临时停车联系页。

扫码后打开页面，点击按钮调用手机原生能力：

- 拨打电话
- 发送短信

手机号当前写死为 `17300987266`。

## 文件

- `index.html`：页面主体
- `.nojekyll`：让 GitHub Pages 直接按静态站点托管

## GitHub Pages 最短部署

1. 新建一个 GitHub 仓库，比如 `move-car-page`
2. 把当前目录里的文件上传到仓库根目录
3. 打开仓库的 `Settings`
4. 打开 `Pages`
5. 在 `Build and deployment` 里选择：
   - `Source`: `Deploy from a branch`
   - `Branch`: `main`
   - `Folder`: `/ (root)`
6. 保存后等待几十秒到几分钟
7. 页面地址通常会变成：
   - `https://你的用户名.github.io/move-car-page/`

## 正确测试方式

不要用电脑本地 `file://` 页面测。

请用下面方式测试：

1. 手机系统相机扫码
2. 打开 GitHub Pages 的 `https://` 地址
3. 在系统浏览器里点击按钮

不建议用这些环境验证：

- 微信内置浏览器
- 支付宝内置浏览器
- 抖音内置浏览器
- 电脑浏览器
